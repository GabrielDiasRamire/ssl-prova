Instalação OpenSSL:

apt update → Atualiza a lista de pacotes disponíveis no sistema.

apt install openssl -y → Instala o OpenSSL sem pedir confirmação (o -y aceita automaticamente).

Verificação de versão
openssl version → Mostra a versão instalada do OpenSSL, confirmando que a instalação foi feita com sucesso.

Explicação do OpenSSL

O OpenSSL é uma biblioteca e ferramenta que implementa protocolos de criptografia, como SSL/TLS, usada para gerar chaves, certificados e proteger comunicações.
