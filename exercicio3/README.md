Explicação:

openssl genrsa → Gera uma chave privada RSA.

-out aluno.key → Salva a chave no arquivo aluno.key.

2048 → Define o tamanho da chave em bits (quanto maior, mais segura; 2048 é o padrão recomendado).

A chave privada é usada para criptografar e assinar dados. Deve ser mantida em segredo, pois garante a identidade e segurança do dono.
